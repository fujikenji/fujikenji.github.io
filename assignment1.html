A1: Blink!

I looked at the available LED colors in the kit and figured that I may as well mimic a traffic light.

Schematic:

The voltage drop across red, yellow, and green LEDs are all 1.8V, and each pin supplies 5V.
5V - 1.8V = 3.2V
And we want to run each LED with 20 mA of current, which is 0.02A
So with V = IR (ohm's law):
3.2V = 0.02A * R
R = 160立 (or more)
I opted to use 220 ohm resistors for each LED because that's the closest stock resistor in the kit that is greater than 160立.
If I used one under 160立, that would be too much current for the LED.

Circuit Image:

A colored wire connects each pin to a 220立 resistor and subsequently each LED, which all runs back to ground.

Code Snippet: 

          
          // Red LED in pin 12
          int red = 12;
          // Yellow LED in pin 10
          int yellow = 10;
          // Green LED in pin 8
          int green = 8;

          // Setup function for the board
          void setup() {
            // Setting the red LED pin (12) to output
            pinMode(red, OUTPUT);
            // Setting the yellow LED pin (10) to output
            pinMode(yellow, OUTPUT);
            // Setting the green LED pin (8) to output
            pinMode(green, OUTPUT);
          }

          // Loop function to control the blinking
          void loop() {
            // Turning on the green light
            digitalWrite(green, HIGH);
            // Waiting 2 seconds
            delay(2000);
            // Turning off green light
            digitalWrite(green, LOW);
            // Turning on yellow light
            digitalWrite(yellow, HIGH);
            // Waiting 1 second
            delay(1000);
            // Turning off yellow light
            digitalWrite(yellow, LOW);
            // Turning on red light
            digitalWrite(red, HIGH);
            // Waiting 4 seconds
            delay(4000);
            // Turning off red light
            digitalWrite(red, LOW);
          }

Blink!

The red light is obscenely long compared to the green and yellow lights
because that's just how red lights seem to feel when I drive - uncomfortably long.
